API Reference
  • 1. Web控制器，提供自动化测试系统的API接口
    &amp;lt;p&amp;gt;
    该控制器提供了测试用例管理、元素库访问和测试用例执行等功能的RESTful接口
    &amp;lt;/p&amp;gt;
    • 1.1. 获取所有测试用例列表

      该接口会扫描测试用例目录下的所有.txt文件，返回测试用例ID和名称
    • 1.2. 获取指定测试用例的详细内容

      读取指定测试用例文件并解析其中的命令步骤，返回结构化的步骤列表
    • 1.3. 获取UI元素库

      从YAML配置文件中读取元素映射配置，返回所有UI元素的列表
    • 1.4. 获取所有可用的动作类型列表
    • 1.5. 获取按context分组的元素列表
    • 1.6. 保存测试用例

      将前端提交的测试步骤保存为标准格式的测试用例文件
    • 1.7. 执行测试用例

      接收前端提交的测试步骤，将其转换为标准命令格式并执行，返回执行结果日志
    • 1.8. 处理图片上传，识别UI元素并更新元素库

      该接口接收上传的图片，调用图像处理服务识别图片中的UI元素，
      更新元素库配置文件，并返回处理结果
    • 1.9. 获取元素对应的图片路径映射

      返回每个元素ID和其对应的图片URL或文件路径的映射关系
    • 1.10. 更新元素信息

      接收前端提交的元素更新信息，更新YAML配置文件中的元素定义
    • 1.11. 删除元素

      从YAML配置文件中删除指定元素
    • 1.12. 获取元素图片

      根据请求参数返回指定元素的图片资源
    • 1.13. 更新元素图片

      接收上传的图片文件，更新指定元素的图片路径
    • 1.14. 创建新元素

      将新元素添加到YAML配置文件中
